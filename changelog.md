# 1.0.0

Clash Lite 是基于原 clash-party 简化和重整后的首个独立版本，定位为轻量、直接、少干预、更尊重原始配置的 Mihomo 桌面客户端。本版本使用独立的软件名称、包名、数据目录、更新源和发布仓库，不兼容也不迁移旧 clash-party 配置。

## 重大变化 (Breaking)

- 软件对外名称改为 Clash Lite，包名、安装包命名、托盘提示、窗口标题、User-Agent、自动更新地址和 GitHub 仓库均切换到 `https://github.com/Yonch404/clash-lite`。
- Clash Lite 作为全新软件发布，不导入、不迁移、不清理改名前旧版本的数据。
- 默认尽量保留订阅原始配置，已移除的功能不会在后台生效，也不会继续改写订阅提供的相关运行时设置。

## 简化移除 (Removed)

- 移除规则管理、外部资源、覆写、DNS 覆写、嗅探覆写、Sub-Store、WebDAV 备份及其入口、设置项和运行时实现。
- 移除“用量”侧边栏和相关统计页面。
- 移除网络信息中的“网络拓扑”部分。
- 移除 Gist 运行时配置同步、指定 Wi-Fi SSID 直连、交换托盘左右键功能。
- 移除 GitHub 下载代理、连接卡片纯数字显示、切换订阅时热重载配置、热重载切换订阅时关闭连接等设置项及相关实现。
- 移除快捷键设置及相关实现。
- 移除第三方主题切换，仅保留默认的深色、浅色、跟随系统主题模式。
- 移除设置页中的官方文档、Telegram 群组入口。
- 移除 TUN 页面中除“重设防火墙”以外的高级配置项。
- 移除内核设置中 Smart 内核、预览版、测试版等额外选择，仅保留非 Smart 内核稳定版选择与更新。
- 移除侧边栏卡片拖动排序、大小调整、隐藏入口及其实现。
- 移除代理组页面顶部的“隐藏不可用节点”和节点排序切换按钮。
- 移除自定义或动态托盘图标切换逻辑，托盘固定使用主图标。

## 行为调整 (Changed)

- 系统代理在首次启动完成配置加载后默认开启，用户仍可手动开关；启动瞬间配置尚未加载时，系统代理和 TUN 均保持关闭显示，避免误导状态。
- TUN 模式默认开启，并在侧边栏保留开关；用户关闭后会尊重用户选择，不会强制重新开启。
- 每个订阅的自动更新默认开启，用户可按订阅单独关闭。
- 切换代理或模式时默认关闭连接，并移除对应设置入口。
- 订阅功能菜单中，“显示二维码”移动到“打开文件”下方。
- 代理组节点顺序改为复刻原版“默认”排序模式。
- 主界面左上角新版本气泡改为显示“更新”，不再直接显示新版版本号，并补充多语言文案。

## 界面体验 (UI)

- 固定侧边栏顺序和尺寸：系统代理(小)、虚拟网卡(小)、代理组(大)、订阅(大)、连接(大)、内核设置(大)、日志(小)、网络信息(小)。
- 优化侧边栏卡片点击体验：普通光标、无缩放动画、按下即选中，仅保留正常、悬停、选中三种背景状态。
- 更新应用主图标并改为静态图标。
- 优化 README，重写为 Clash Lite 正式项目说明。

## 维护 (Chore)

- 整理项目包信息和发布信息，更新依赖并升级 Electron。
- 清理不再使用的图标、文档和已移除功能相关代码路径。
- 保留静态检查流程，用于验证魔改后的类型、格式和基础代码质量。
